# 卸載

本頁介紹從設備上移除 Battman 的一般方法。

!!! note "注意"
    如果您之前進行了系統級更改（溫控調校等），請盡可能在卸載前將它們配置為默認值。

## 通過 TrollStore 安裝

- 打開 TrollStore
- 找到並點擊 "Battman"
- 點擊 "Uninstall App" 以卸載 Battman

## 通過包管理器安裝

如果 Battman 是通過包管理器安裝的（例如，在越獄設備上）：

- 首先打開 Battman 並清除 Battman 數據（請參閱**清除數據**）。
- 打開您的包管理器（Cydia、Sileo、Zebra 等）。
- 按名稱找到 Battman 軟件包。
- 使用標準的**移除**/**卸載**操作。
