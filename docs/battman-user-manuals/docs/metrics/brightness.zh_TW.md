# 亮度

**亮度**頁面會讀取並展示屏幕亮度數據。

它對應「電池」標籤中的**亮度**區域。

## 顯示內容

- 當前亮度（以 **用戶百分比** 表示）。
- **nits** 亮度上下限（硬件與用戶可訪問值）。
- 屏幕特性：分辨率、色域（sRGB / P3）、刷新率、色彩深度等。

## 高級控制

在**高級**中，我們目前提供了有限的系統全局配置控制：

- **低電量模式亮度減弱**：控制系統在低電量模式啓用時的屏幕顯示亮度的減弱程度。
- **自動調暗**：控制系統在設備閒置時是否自動調暗。
- **接入電源時自動調暗**：類似**自動調暗**，但只控制接入電源後的行為。

### 百分比與尼特並非線性對應

- iOS 會用感知（類似 gamma）的曲線把用戶滑塊映射到面板輸出，而不是直線。
- 該函數大致為指數型：`f(百分比) ≠ 最大nits * 百分比`。
- 例如，在 650nit 面板上，50% 的用戶亮度通常只有 ~150 nit，而不是 ~325。
- 這種曲線讓低/中亮度對人眼更舒適。

## 說明

- 通過 Battman 修改**高級**配置會直接更改系統的全局亮度設置。iOS 會將這些設置視為用戶數據，備份時會被保存；如果將備份恢復到另一台設備，同樣的亮度設置也會隨之應用。
