# 亮度

**亮度**页面会读取并展示屏幕亮度数据。

它对应「电池」标签中的**亮度**区域。

## 显示内容

- 当前亮度（以 **用户百分比** 表示）。
- **nits** 亮度上下限（硬件与用户可访问值）。
- 屏幕特性：分辨率、色域（sRGB / P3）、刷新率、色彩深度等。

## 高级控制

在**高级**中，我们目前提供了有限的系统全局配置控制：

- **低电量模式亮度减弱**：控制系统在低电量模式启用时的屏幕显示亮度的减弱程度。
- **自动调暗**：控制系统在设备闲置时是否自动调暗。
- **接入电源时自动调暗**：类似**自动调暗**，但只控制接入电源后的行为。

### 百分比与尼特并非线性对应

- iOS 会用感知（类似 gamma）的曲线把用户滑块映射到面板输出，而不是直线。
- 该函数大致为指数型：`f(百分比) ≠ 最大nits * 百分比`。
- 例如，在 650nit 面板上，50% 的用户亮度通常只有 ~150 nit，而不是 ~325。
- 这种曲线让低/中亮度对人眼更舒适。

## 说明

- 通过 Battman 修改**高级**配置会直接更改系统的全局亮度设置。iOS 会将这些设置视为用户数据，备份时会被保存；如果将备份恢复到另一台设备，同样的亮度设置也会随之应用。
