# 电池信息

**电池信息**页面展示的是直接从电量监测计（Gas Gauge）和`IOAccessoryManager`读取的详细数据。

它对应「电池」标签的**电池信息**区域。

## 你可以看到

- 当前电量百分比、电压、电流
- 设计容量、标称/实际满充容量以及循环次数
- 充电状态与电源类型（USB、适配器、无线、MagSafe 等）
- 未充电原因

与依赖 `IOPM` 的其他工具不同，Battman **不会**猜测或美化数值：它尽量原样呈现硬件报告的最底层原始数据。

## 板块

在「电池信息」页面上，你可能会看到以下部分（取决于设备当前状态）：

- [电量监测计（基本）](#_4)
- [适配器信息](#_14)
- [非接触式端口 / 串行接口](#_20)

每个部分会列出来自不同模块的已检测指标。

### 电量监测计（基本）

此部分展示最常用的电量监测计数据。

#### 设备信息

| 字段 | 说明 |
| --- | --- |
| 设备名称 | 已安装电池使用的电量计 IC（如 `bq20z451`）。 |
| 电池序列号 | 电池的序列号。 |
| 化学 ID | 用于匹配系统中电池特定策略的 ChemID。 |
| 电芯数量 | 电池的电芯数量（例如 `3` 表示三芯电池）。 |

#### 容量指标

| 字段 | 说明 |
| --- | --- |
| 满充容量 | 当前由电量计 IC 报告的最大可充容量（mAh）。 |
| 设计容量 | 电池出厂时的原始设计容量（mAh）。 |
| 剩余容量 | 当前剩余可用容量（mAh）。 |
| 最大化学容量 | 电池设计的最大化学容量（Qmax）。 |
| 真实剩余容量 | 电池管理系统（BMS）报告的实际剩余容量（mAh）。 |
| 电池运行时间 | 自上次复位以来电池已工作时间。 |

#### 放电指标

| 字段 | 说明 |
| --- | --- |
| 放电深度 (DOD₀) | 处于 `relaxed` 状态时由开路电压更新的化学放电深度。 |
| 充放累积 | 当前充放电周期的累积容量；每次 DOD₀ 更新时重置。 |

#### 电气测量

| 字段 | 说明 |
| --- | --- |
| 电压 | 平均工作电压，单位 mV（如 `11433 mV`）。 |
| 平均电流 | 平均工作电流，单位 mA（负值=放电，如 `-807 mA`）。 |
| 平均功率 | 平均工作功率，单位 mW（负值=消耗，如 `-9226 mW`）。 |
| 开路电流 | 开路（OCV）电压电流测量值。 |
| 开路电压 | 开路（OCV）电压测量值。 |
| 最大负荷电流 | 电池可承受的最大负载电流。 |
| 最大负荷电流2 | 第二组最大负载电流测量。 |

#### 电量状态

| 字段 | 说明 |
| --- | --- |
| 充电状态 | 电量计芯片报告的电池百分比。 |
| 充电状态（UI） | UI 显示的 SoC（与实际 SoC 不同）。 |
| 单日最大/最小余量 | 电池管理系统报告的一天内达到的最高/最低电量百分比。 |

#### 时间预估

| 字段 | 说明 |
| --- | --- |
| 满充时间 | 预计达到满电所需时间。 |
| 耗尽时间 | 预计耗尽所需时间。 |

#### 循环信息

| 字段 | 说明 |
| --- | --- |
| 循环计数 | 已完成的充放电循环次数（如 `1145`）。 |
| 设计循环次数 | 电池设计可承受的最大循环次数。 |

#### 温度

| 字段 | 说明 |
| --- | --- |
| 平均温度 | 电量计芯片报告的平均工作温度（如 `30.73 °C`）。 |

#### 高级指标

| 字段 | 说明 |
| --- | --- |
| 阻抗系数 | 用于电阻计算的缩放因子。 |
| 标示位 | 参见[标示位](../../troubleshooting/battery-info/flags)。 |
| IT 杂项数据 | Impedance Track 的其他状态信息。 |
| 模拟频率 | 电量计芯片执行 Impedance Track™ 模拟的速率。 |

### 适配器信息

此部分展示当前工作适配器<sup>\[1\]</sup>的信息。

!!! note "当前工作适配器"
    当连接一个或多个适配器时，设备会选择最佳的适配器作为工作电源；其他适配器将被抑制。

#### 连接信息

| 字段 | 说明 |
| --- | --- |
| 端口 | 正在使用的端口 ID。iPhone/iPad：`1` 串行接口（Lightning/USB‑C），`2` 非接触式（MagSafe/无线）。MacBook：`1` USB 端口 1，`2` USB 端口 2。 |
| 适配器类型 | 描述当前适配器类型的数值 ID（未公开文档）。 |
| 类型 | 已连接适配器的类型代码（`kIOPSPowerAdapterFamilyKey`）。参见[类型代码](../../troubleshooting/battery-info/family-keys)。 |

#### 充电状态

| 字段 | 说明 |
| --- | --- |
| 状态 | 充电状态：`正在充电` 或 `不在充电`。 |
| 原因 | 未充电时的原因代码。参见[未充电原因](../../troubleshooting/battery-info/not-charging-reason)。 |
| 充电器兼容 | 适配器被识别为兼容的电源。 |
| 已连接电源 | 适配器存在且可作为稳定电源使用。 |

#### 电力规格

| 字段 | 说明 |
| --- | --- |
| 额定电流 / 额定电压 | 适配器提供的设计电流/电压。 |
| 输入电流 / 输入电压 | 实际接收到的输入电流/电压。 |
| 充电电流 / 充电电压 | 实际给电池充电的电流/电压。 |
| PMU 配置 / 充电器配置 | PMU/充电器设置的充电电流上限。 |
| 高压充电模式 | 适配器硬件支持的高压充电模式（但用户不可配置）。 |

#### 硬件信息

| 字段 | 说明 |
| --- | --- |
| 充电芯片ID | 设备充电芯片的数值 ID。一些 USB‑C 设备上 `0xFFFFFFFF` 表示无法读取ID。 |

#### 适配器识别

以下字段仅在适配器为苹果设备设计（或模拟 Apple/MFi 原装适配器）时可用：

| 字段 | 说明 |
| --- | --- |
| 型号名称 | 适配器型号名称（如 “61W USB-C Power Adapter”）。 |
| 生产企业 | 厂商名称（如 “Apple Inc.”）。 |
| 型号号码 | 型号号码（如 `0x1685`）。 |
| 固件版本 | 固件版本（如 `01050029` → `1.5.0.29`）。 |
| 硬件版本 | 硬件版本（如 `1.0`）。 |
| 序列号 | 适配器序列号（如 `C4H306600XDL4YRAD`）。 |
| 描述 | 电源管理单元（PMU）对适配器类型的描述（如 “pd charger”）。 |

### 非接触式端口 / 串行接口

当设备连接到提供信息的非接触式电源配件 / USB 配件时，本板块会出现。

#### 识别信息

| 字段 | 说明 |
| --- | --- |
| 配件ID | 系统内部使用的已连接配件的数值 ID。 |
| 数字ID | （仅 Lightning）配件的芯片 ID。 |
| 设备序列号 | （仅 Lightning）接口设备序列号（ID‑SN）。 |
| 模块序列号 | （仅 Lightning）接口模块序列号（MSN）。 |
| PPID | 配件 PPID（厂商提供的标识符）。 |
| 端口类型 | 来自 `IOAccessoryManager` 的连接端口类型。 |
| 允许的特性 | 配件功能标示位（可能因设备而异）。 |
| 序列号 | （仅 MagSafe）配件序列号。 |
| 生产企业 | 配件厂商。 |
| 产品ID | HID PID。 |
| 厂商ID | HID VID。 |
| 型号 | 配件型号（如 `A2384`）。 |
| 名称 | 配件名称（如 “MagSafe Battery Pack”）。 |
| 固件版本 | 配件固件版本（如 `2.7.b.0`）。 |
| 硬件版本 | 配件硬件版本（如 `1.0.0`）。 |

#### 供电角色

| 字段 | 说明 |
| --- | --- |
| 外接电池 | 指示配件是否作为[Smart Battery Case](https://apple.fandom.com/wiki/Smart_Battery_Case)。 |
| 正在提供电源 | 配件正作为电源工作。 |
| 状态 | （仅 MagSafe）当前配件工作状态。 |
| 正在接受充电 | （仅配件电池）配件是否正在接受充电。 |

#### 配件电池（UPS）

| 字段 | 说明 |
| --- | --- |
| 剩余容量 | 配件电池百分比。 |
| 满充容量 | 配件电池满充容量。 |
| 当前容量 | 配件电池当前容量。 |
| 平均充电电流 | 接受充电时的平均充电电流。 |
| 额定充电电压 | 充电额定电压。 |
| 电芯数量 | 配件电池的电芯数量。 |
| 循环计数 | 配件电池的循环次数。 |
| 电流 / 电压 | 配件当前输出电流/电压。 |
| 输入电流 / 输入电压 | 设备接收到的实时电流/电压。 |
| 温度 | 当前配件温度。 |
| 配件满充时间 / 配件耗尽时间 | 配件电量计估算的时间。 |
| 电源模式 | 配件的活动电源模式。 |
| 睡眠电源 | 配件的休眠电源模式。 |
| 受监管配件已连接 | 配件是否被标记为**受监管配件**。 |
| 监管传输已受限 | 受监管传输是否受限制。 |

#### USB（串行接口）

| 字段 | 说明 |
| --- | --- |
| USB 连接状态 | 当前 USB 设备连接状态。 |
| USB 充电电压 | 当前 USB 设备充电电压。 |
| USB 电流配置 | 当前 USB 设备充电电流配置。 |
