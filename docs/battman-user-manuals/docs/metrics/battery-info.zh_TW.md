# 電池信息

**電池信息**頁面展示的是直接從電量監測計（Gas Gauge）和`IOAccessoryManager`讀取的詳細數據。

它對應「電池」標籤的**電池信息**區域。

## 你可以看到

- 當前電量百分比、電壓、電流
- 設計容量、標稱/實際滿充容量以及循環次數
- 充電狀態與電源類型（USB、適配器、無線、MagSafe 等）
- 未充電原因

與依賴 `IOPM` 的其他工具不同，Battman **不會**猜測或美化數值：它盡量原樣呈現硬件報告的最底層原始數據。

## 板塊

在「電池信息」頁面上，你可能會看到以下部分（取決於設備當前狀態）：

- [電量監測計（基本）](#_4)
- [適配器信息](#_14)
- [非接觸式端口 / 串行接口](#_20)

每個部分會列出來自不同模塊的已檢測指標。

### 電量監測計（基本）

此部分展示最常用的電量監測計數據。

#### 設備信息

| 字段 | 說明 |
| --- | --- |
| 設備名稱 | 已安裝電池使用的電量計 IC（如 `bq20z451`）。 |
| 電池序列號 | 電池的序列號。 |
| 化學 ID | 用於匹配系統中電池特定策略的 ChemID。 |
| 電芯數量 | 電池的電芯數量（例如 `3` 表示三芯電池）。 |

#### 容量指標

| 字段 | 說明 |
| --- | --- |
| 滿充容量 | 當前由電量計 IC 報告的最大可充容量（mAh）。 |
| 設計容量 | 電池出廠時的原始設計容量（mAh）。 |
| 剩餘容量 | 當前剩餘可用容量（mAh）。 |
| 最大化學容量 | 電池設計的最大化學容量（Qmax）。 |
| 真實剩餘容量 | 電池管理系統（BMS）報告的實際剩餘容量（mAh）。 |
| 電池運行時間 | 自上次復位以來電池已工作時間。 |

#### 放電指標

| 字段 | 說明 |
| --- | --- |
| 放電深度 (DOD₀) | 處於 `relaxed` 狀態時由開路電壓更新的化學放電深度。 |
| 充放累積 | 當前充放電週期的累積容量；每次 DOD₀ 更新時重置。 |

#### 電氣測量

| 字段 | 說明 |
| --- | --- |
| 電壓 | 平均工作電壓，單位 mV（如 `11433 mV`）。 |
| 平均電流 | 平均工作電流，單位 mA（負值=放電，如 `-807 mA`）。 |
| 平均功率 | 平均工作功率，單位 mW（負值=消耗，如 `-9226 mW`）。 |
| 開路電流 | 開路（OCV）電壓電流測量值。 |
| 開路電壓 | 開路（OCV）電壓測量值。 |
| 最大負荷電流 | 電池可承受的最大負載電流。 |
| 最大負荷電流2 | 第二組最大負載電流測量。 |

#### 電量狀態

| 字段 | 說明 |
| --- | --- |
| 充電狀態 | 電量計芯片報告的電池百分比。 |
| 充電狀態（UI） | UI 顯示的 SoC（與實際 SoC 不同）。 |
| 單日最大/最小餘量 | 電池管理系統報告的一天內達到的最高/最低電量百分比。 |

#### 時間預估

| 字段 | 說明 |
| --- | --- |
| 滿充時間 | 預計達到滿電所需時間。 |
| 耗盡時間 | 預計耗盡所需時間。 |

#### 循環信息

| 字段 | 說明 |
| --- | --- |
| 循環計數 | 已完成的充放電循環次數（如 `1145`）。 |
| 設計循環次數 | 電池設計可承受的最大循環次數。 |

#### 溫度

| 字段 | 說明 |
| --- | --- |
| 平均溫度 | 電量計芯片報告的平均工作溫度（如 `30.73 °C`）。 |

#### 高級指標

| 字段 | 說明 |
| --- | --- |
| 阻抗系數 | 用於電阻計算的縮放因子。 |
| 標示位 | 參見[標示位](../../troubleshooting/battery-info/flags)。 |
| IT 雜項數據 | Impedance Track 的其他狀態信息。 |
| 模擬頻率 | 電量計芯片執行 Impedance Track™ 模擬的速率。 |

### 適配器信息

此部分展示當前工作適配器<sup>\[1\]</sup>的信息。

!!! note "當前工作適配器"
    當連接一個或多個適配器時，設備會選擇最佳的適配器作為工作電源；其他適配器將被抑制。

#### 連接信息

| 字段 | 說明 |
| --- | --- |
| 端口 | 正在使用的端口 ID。iPhone/iPad：`1` 串行接口（Lightning/USB‑C），`2` 非接觸式（MagSafe/無線）。MacBook：`1` USB 端口 1，`2` USB 端口 2。 |
| 適配器類型 | 描述當前適配器類型的數值 ID（未公開文檔）。 |
| 類型 | 已連接適配器的類型代碼（`kIOPSPowerAdapterFamilyKey`）。參見[類型代碼](../../troubleshooting/battery-info/family-keys)。 |

#### 充電狀態

| 字段 | 說明 |
| --- | --- |
| 狀態 | 充電狀態：`正在充電` 或 `不在充電`。 |
| 原因 | 未充電時的原因代碼。參見[未充電原因](../../troubleshooting/battery-info/not-charging-reason)。 |
| 充電器兼容 | 適配器被識別為兼容的電源。 |
| 已連接電源 | 適配器存在且可作為穩定電源使用。 |

#### 電力規格

| 字段 | 說明 |
| --- | --- |
| 額定電流 / 額定電壓 | 適配器提供的設計電流/電壓。 |
| 輸入電流 / 輸入電壓 | 實際接收到的輸入電流/電壓。 |
| 充電電流 / 充電電壓 | 實際給電池充電的電流/電壓。 |
| PMU 配置 / 充電器配置 | PMU/充電器設置的充電電流上限。 |
| 高壓充電模式 | 適配器硬件支持的高壓充電模式（但用戶不可配置）。 |

#### 硬件信息

| 字段 | 說明 |
| --- | --- |
| 充電芯片ID | 設備充電芯片的數值 ID。一些 USB‑C 設備上 `0xFFFFFFFF` 表示無法讀取ID。 |

#### 適配器識別

以下字段僅在適配器為蘋果設備設計（或模擬 Apple/MFi 原裝適配器）時可用：

| 字段 | 說明 |
| --- | --- |
| 型號名稱 | 適配器型號名稱（如 「61W USB-C Power Adapter」）。 |
| 生產企業 | 廠商名稱（如 「Apple Inc.」）。 |
| 型號號碼 | 型號號碼（如 `0x1685`）。 |
| 固件版本 | 固件版本（如 `01050029` → `1.5.0.29`）。 |
| 硬件版本 | 硬件版本（如 `1.0`）。 |
| 序列號 | 適配器序列號（如 `C4H306600XDL4YRAD`）。 |
| 描述 | 電源管理單元（PMU）對適配器類型的描述（如 「pd charger」）。 |

### 非接觸式端口 / 串行接口

當設備連接到提供信息的非接觸式電源配件 / USB 配件時，本板塊會出現。

#### 識別信息

| 字段 | 說明 |
| --- | --- |
| 配件ID | 系統內部使用的已連接配件的數值 ID。 |
| 數字ID | （僅 Lightning）配件的芯片 ID。 |
| 設備序列號 | （僅 Lightning）接口設備序列號（ID‑SN）。 |
| 模塊序列號 | （僅 Lightning）接口模塊序列號（MSN）。 |
| PPID | 配件 PPID（廠商提供的標識符）。 |
| 端口類型 | 來自 `IOAccessoryManager` 的連接端口類型。 |
| 允許的特性 | 配件功能標示位（可能因設備而異）。 |
| 序列號 | （僅 MagSafe）配件序列號。 |
| 生產企業 | 配件廠商。 |
| 產品ID | HID PID。 |
| 廠商ID | HID VID。 |
| 型號 | 配件型號（如 `A2384`）。 |
| 名稱 | 配件名稱（如 「MagSafe Battery Pack」）。 |
| 固件版本 | 配件固件版本（如 `2.7.b.0`）。 |
| 硬件版本 | 配件硬件版本（如 `1.0.0`）。 |

#### 供電角色

| 字段 | 說明 |
| --- | --- |
| 外接電池 | 指示配件是否作為[Smart Battery Case](https://apple.fandom.com/wiki/Smart_Battery_Case)。 |
| 正在提供電源 | 配件正作為電源工作。 |
| 狀態 | （僅 MagSafe）當前配件工作狀態。 |
| 正在接受充電 | （僅配件電池）配件是否正在接受充電。 |

#### 配件電池（UPS）

| 字段 | 說明 |
| --- | --- |
| 剩餘容量 | 配件電池百分比。 |
| 滿充容量 | 配件電池滿充容量。 |
| 當前容量 | 配件電池當前容量。 |
| 平均充電電流 | 接受充電時的平均充電電流。 |
| 額定充電電壓 | 充電額定電壓。 |
| 電芯數量 | 配件電池的電芯數量。 |
| 循環計數 | 配件電池的循環次數。 |
| 電流 / 電壓 | 配件當前輸出電流/電壓。 |
| 輸入電流 / 輸入電壓 | 設備接收到的實時電流/電壓。 |
| 溫度 | 當前配件溫度。 |
| 配件滿充時間 / 配件耗盡時間 | 配件電量計估算的時間。 |
| 電源模式 | 配件的活動電源模式。 |
| 睡眠電源 | 配件的休眠電源模式。 |
| 受監管配件已連接 | 配件是否被標記為**受監管配件**。 |
| 監管傳輸已受限 | 受監管傳輸是否受限制。 |

#### USB（串行接口）

| 字段 | 說明 |
| --- | --- |
| USB 連接狀態 | 當前 USB 設備連接狀態。 |
| USB 充電電壓 | 當前 USB 設備充電電壓。 |
| USB 電流配置 | 當前 USB 設備充電電流配置。 |
