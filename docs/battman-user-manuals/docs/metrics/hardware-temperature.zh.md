# 硬件温度

**硬件温度**页面展示设备各类温度传感器的实时读数。

它对应「电池」标签中的**硬件温度**区域。

## 分区

Battman 会将温度数据分为以下板块：

- **系统温控状态**  
  显示系统温控值守进程 `thermalmonitord` 的状态。
- **温控信息**  
  上层热指标：温控压力等级、[温控通知等级](../../troubleshooting/thermals/thermal-notification-levels)、最大触发温度、阳光暴晒状态。
- **设备传感器**  
  系统报告的逐个传感器读数（如 SoC、机身、电池模组、摄像头模组）。
- **HID / HID 原始数据**  
  来自多个硬件传感器的底层 HID 温度。

## 使用说明

- 部分传感器可能不会实时更新，或只返回占位值。
- 热行为可在**温控调校**中调整；本页专注只读监控。
