# 「未知」屏幕溫度

有時 Battman 無法將系統的讀數映射到有意義的屏幕或面板溫度。在這種情況下，界面可能會在屏幕溫度處顯示**「未知」**，或者完全省略該數值。

## 可能原因

- 設備固件並未提供單獨的面板溫度傳感器。
- 當前 iOS 版本隱藏或聚合了相關傳感器。
- Battman 運行在模擬器或不受支持的平台（例如某些 Mac）。
- Battman 尚未支持讀取您設備的面板溫度。

## 可以怎麼處理

- 使用**硬件溫度**頁面查看其他可用傳感器。
- 檢查異常的熱壓或降頻現象，而不是依賴單一的屏幕溫度數值。
- 保持 Battman 為最新版本；可能會隨著時間的推移添加對更多傳感器的支持。
