# 亮度後端

Battman 會區分系統顯示中使用的不同**亮度控制後端**。

- **標準** – 傳統的背光控制路徑，多見於早期或結構較簡單的設備。
- **DCP** – 基於 Display Co‑Processor（顯示協處理器）的管線，用於較新的（A14 或更高）、面板控制更複雜的設備。

## 這意味著什麼

- 不同後端暴露的亮度上限、變化曲線以及響應延遲可能不同。
- 某些高級特性（如 HDR、更高刷新率）往往依賴 DCP 管線。
- 部分缺陷或奇怪行為可能僅在特定後端上出現，瞭解當前後端有助於問題排查。

此字段僅供參考，Battman 本身不會切換亮度後端。
