# 警告條件

本頁說明 Battman 基於電池與系統報告的數據而發出警告的條件。

## **電池信息**中的數據警告

Battman 會標出可疑或丟失的**電池信息**監測數據，並在此列的側邊顯示一個「警告」圖標，輕按圖標可以顯示如下之一的詳細信息。

- **剩餘容量**
    - 當其報告數據超過了「滿充容量」，或者比設備計算的「真實剩餘容量」多出大於 10 mAh 時，報告異常數據信息：「電池剩餘容量數據異常，使用的可能是未經認證的電池部件。」，並且在其超過「滿充容量」時，警告信息中會顯示一個基於現有數據計算的「預估剩餘」。
    - 當數據丟失時，觸發「未獲取到剩餘容量。」
- **循環計數**
    - 當「循環計數」超出「設計循環次數」時警告；當電量計未提供「設計循環次數」時，Battman 將會根據 Apple 官方文檔進行推測（iPhone 15 之前：500；之後的 iPhone 與 iPad、Watch、MacBook: 1000；iPod：400）。警告信息：「已循環超過設計標準的週期次數，建議更換原廠電池組件。」
- **耗盡時間**
    - 當設備不在充電時，Battman 將報告的「耗盡時間」與理想預估（容量 ÷ 耗電電流）進行對比。如果報告的「耗盡時間」比理想預估多出了 1.5 倍，則警告「電池耗盡時間異常，使用的可能是未經認證的電池部件」。
- **放電深度**
    - 當「放電深度」比 3 倍的「化學最大容量」還要大時，數據被認為不典型（即便一直在充電），警告「電池化學放電深度異常，使用的可能是未經認證的電池部件。」

警告標題視特定條件可能為其中之一：「錯誤的數據」、「異常的數據」、「過大的數據」或者「空數據」。

如果你持續遇到這些警告，請考慮尋求專業硬件診斷。
