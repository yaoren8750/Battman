# 標示位

Battman 顯示由電池電量計芯片直接上報的原始標示位，這些位來自德州儀器的電量計（bq274xx、bq275xx、bq20zxx），在**標示位**行按高字節和低字節各一行呈現。

## 在 App 里你會看到

- 一個十六進制數（如 `0x00CC`）表示完整的 16 位標示字。
- 兩行分段開關顯示各位：上行為高字節，下行為低字節。
- 高亮/填充的段表示該位被置位，灰色表示未置位。

## Battman 如何映射標示位

- 若能識別具體電量計型號（設備名或主板標識），Battman 會加載對應的佈局；否則會選擇最接近的佈局。
- 兩行展示全部位，點亮的段即為置位。
- 含義遵循德州儀器的數據手冊，核心字段見下表。

## 說明

- 標示位佈局遵循 TI 文檔；閾值和時序請參考對應數據手冊。
- Battman 僅展示這些位；解讀依賴充電狀態與整機配置。如標示位長期與實際不符，可能是電池包或電量計上報異常。
