# 清除數據

本頁說明如何清除 Battman 的本地數據與設置。

## 這裡的「數據」包括什麼

- Battman 自身的偏好設置（存儲在應用容器或系統設置中）。
- 應用創建的緩存日誌或臨時文件。

!!! note "重要"
    Battman 出於穩定性考慮，不會重置系統級配置（溫控調校等），請盡可能自行備份。

## 清除數據的步驟

1. 打開 Battman
2. 點擊「更多」標籤頁。
3. 點擊「設置」。
4. 滾動到底部，您將看到**抹除全部 Battman 數據**。
5. 點擊它並確認以清除所有 Battman 數據。

如需卸載 Battman，請參閱**卸載**。
