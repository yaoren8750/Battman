# 清除数据

本页说明如何清除 Battman 的本地数据与设置。

## 这里的「数据」包括什么

- Battman 自身的偏好设置（存储在应用容器或系统设置中）。
- 应用创建的缓存日志或临时文件。

!!! note "重要"
    Battman 出于稳定性考虑，不会重置系统级配置（温控调校等），请尽可能自行备份。

## 清除数据的步骤

1. 打开 Battman
2. 点击「更多」标签页。
3. 点击「设置」。
4. 滚动到底部，您将看到**抹除全部 Battman 数据**。
5. 点击它并确认以清除所有 Battman 数据。

如需卸载 Battman，请参阅**卸载**。
