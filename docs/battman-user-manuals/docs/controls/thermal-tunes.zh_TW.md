# 溫控調校

**溫控調校**頁面公開了一些通常不會向用戶展示的底層溫控管理開關。

## 分區說明

- **通用**
  控制應用是否接收熱狀態更新，以及系統在發熱時是否主動降低功耗。
- **口袋高溫模式（HIP）**
  在支持的 iPhone 上，控制屏幕熄滅且設備放入口袋時的溫控策略。
- **陽光暴曬**
  在自動檢測與手動模式之間切換，並展示當前的陽光暴露狀態。
- **溫控等級**
  展示並允許手動調整當前的溫控壓力與通知等級，通過彩色分段條進行控制。

## 開關說明

- **🔒鎖定圖標**：阻止 `thermalmonitord` 將該設置恢復為默認值。當你需要讓自定義值在重啓或系統調整後繼續生效時開啓。
- **✅勾選開關**：功能本身的開/關狀態。根據需要而啓用或關閉具體行為。

## 安全提示

修改這些設置可能會：

- 提升機身與電池溫度；
- 影響充電、背光、手電筒與多媒體行為。

如非充分瞭解風險，請謹慎使用。

## 說明

- 通過 Battman 修改**溫控調校**配置會直接更改系統的全局溫控設置。iOS 會將這些設置視為用戶數據，備份時會被保存；如果將備份恢復到另一台設備，同樣的溫控設置也會隨之應用。
