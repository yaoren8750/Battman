# 充電管理

**充電管理**頁面用於調節系統的充電行為以及低電量模式的自動化邏輯。

## 功能概覽

- **阻止充電**  
  通過訪問電量計 / SMC 寄存器，停止對電池充電，同時繼續使用外接電源為設備供電。
- **阻止外部供電**  
  即便連接電源，也強制優先從電池放電，便於進行受控的循環充放電。
- **優化電池充電**  
  與 `PowerUI` 集成，可安排優化充電時段（起始和結束時間）。
- **低電量模式**  
  打開 / 關閉低電量模式，並根據插電狀態和電量閾值自動退出，可選擇隱藏系統低電量提示。


## 說明

- 通過 Battman 修改**低電量模式**配置會直接改寫系統的全局低電量模式設置。iOS 會將這些設置視為用戶數據，備份時會被保存；如果將備份恢復到另一台設備，同樣的低電量模式設置也會隨之應用。
