# Battman

歡迎使用 Battman 用戶手冊。

Battman 是一款面向 AppleSMC 設備的電源管理工具。通過直接訪問系統管理控制器（SMC），Battman 可以提供專業級的電池健康、充電循環、溫度與功耗指標。借助深入的硬件集成，你可以自定義充電行為、設置電量閾值以延長電池壽命，並在單一界面中實時監控狀態。

### 主要特性

- 專業級數據：查看電壓、電流、容量、循環次數、溫度等詳細信息。
- 自定義充電策略：定義開始 / 停止充電的電量百分比，從而在長期使用中保護電池。
- 實時監控：實時觀察電源與溫度變化情況，並可按需記錄日誌。
- 輕量且開源：佔用資源小，源碼開放，便於審閱與貢獻。

無論你是開發者、普通用戶，還是 IT 管理員，Battman 都能幫助你做出更聰明、更高效的用電決策。
