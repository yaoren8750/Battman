# 在 macOS 上安裝

目前 **尚未提供 Battman 的 macOS 原生界面**，macOS 支持仍在規劃中。

當前 Battman 主要面向越獄設備或通過 TrollStore 安裝的 iOS 設備。一些底層組件和庫可以在 macOS / Apple 芯片 Mac 上編譯用於開發和調試，但暫時沒有面向普通用戶的 macOS 圖形界面應用。

## 當前狀態

- 暫無官方的 macOS 應用包或安裝器。
- 某些核心庫可以在 macOS 11+ 上編譯，僅供開發使用。
- 與電池、溫度相關的 macOS API 仍在調研中。

## 當 macOS 版本準備就緒時

本頁將補充說明：

- 系統要求（支持的 macOS 版本與機型）。
- 如何獲取並安裝 Battman for macOS（下載、校驗與運行方式）。
- 沙盒、權限與數據目錄的注意事項。

在此之前，如需實際使用，請參考 iOS 安裝章節。
